1. A web-based emergency alert application designed to enhance women's safety.
2. It allows users to send real-time alerts along with their live location.
3. One-click SOS alert button and one-click cancel button to cancel alert.
4. Sends Name , Contact , SMS , Audio recorder and Live Location to the Server.
5. Stores user details securily in a database.
6. Simple and Responsive user interface.
7. Used HTML , CSS , Javascript , MediaRecorder API , Geolocation API.
8. Used Node.js , MongoDB , Express.js , OpenStreetMap Nominatim.

10. *Backend Code: Available in backend branch.

# Emergency-Alert-Web-App-for-Women-Safety
A web-based emergency alert application designed to enhance women's safety. It allows users to send real-time alerts along with their live location using latitude and longitude.
One-click SOS alert button and one-click cancel button to cancel alert.
Sends Name , Contact , SMS , Audio recorder and Live Location to the Server.
Stores user details securily in a database.
Simple and Responsive user interface.
Used tech-stack(HTML , CSS , Javascript , mongoDB and Node.js).
